# 다이나믹 프로그래밍

by 나동빈





* 메모리를 적절히 사용해서 수행 시간 효율성을 비약적으로 향상시키는 방법

* 이미 계산된 결과를 별도의 메모리 영역에 저장하여 다시 계산하는 것을 방지

* 두 가지 방식(탑다운, 바텀업)으로 구성

* 동적 계획법이라고도 부른다
* 다이나믹 프로그래밍의 '다이나믹'은 큰 의미 없이 사용된 단어
